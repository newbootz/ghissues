== DESCRIPTION:

CLI app to interact with GitHub 'issues' API

== FEATURES/PROBLEMS:

Quite new.  Does some useful stuff.  


== SYNOPSIS:

  ghissues open
      Lists open issues

  
  ghissues create <title> <body>
      Creates new issue
     
       
== REQUIREMENTS:

Ruby.  I think that's all.


== INSTALL:

The usual gem install stuff
